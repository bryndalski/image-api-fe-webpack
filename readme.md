# Image api



# # How to bootstrap

### 1. Install the package

```agsl
npm install
```

### 2. Set up the environment variables

```agsl
touch .env
```

### 3. Add the following environment variables to the .env file

```agsl
COGNITO_CLIENT_ID=
COGNITO_USER_POOL_ID=
```


### 4. Run the project

```agsl
npm start
```

Project will be running on `http://localhost:4000`
